VectorBase

VectorBase is one of the base mathematical classes in open source library COPT. VectorBase describes a basic type vector. A vector is a multiple dimensional 'Array' with a number of arithmetic operations including summation, subtraction, multiplication etc. This class derives directly from another two basic type called Array and AbstractVector. It takes scalar, index and SizeAtCompileTime as template.

VectorBase(const index size, const referred_array& tag, scalar *data, const index inter = 1)
Constructor of a referred 'VectorBase'. Referred 'VectorBase' is a special kind of 'VectorBase' which does not have its own data. The data is stored somewhere else. As an instance, it is not convenient to construct another 'VectorBase' when representating a column or a row of a 'MatrixBase'. But it is efficient just referred to the data. Here inter is the interval between two elements.

Parameters:

size: The size of the 'VectorBase'

tag: The trait of referred 'VectorBase'. A consideration of C++ feature.

data: The pointer to the array of data

inter: The interval between two elements. '1' is used as default.
