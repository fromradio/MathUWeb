SpMatrixBase

SpMatrixBase is one of the base mathematical classes in open source library COPT. SpMatrixBase describes a basic type sparse matrix. This class takes scalar and index as template.

SpMatrixBase()
This is the default constructor of the SpMatrixBase. A 0-by-0 sparse matrix will be generated as default.
