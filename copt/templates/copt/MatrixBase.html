MatrixBase

MatrixBase is one of the base mathematical classes in open source library COPT. MatrixBase describes a basic type matrix. A matrix is a multiple dimensional 'Array' with a number of arithmetic operations including summation, subtraction, multiplication etc. This class derives directly from another two basic type called Array and AbstractMatrix. It takes scalar, index, RowAtCompileTime and ColAtCompileTime as template.
